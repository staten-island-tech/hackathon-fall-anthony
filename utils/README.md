# This folder is used for development purposes only.

These files are not needed for the product and are only used to facilitate processes done during development.

* [ ] `add_ids.py` - Adds the `id` attribute to every note.
* [ ] `backup_and_clean_notes.py` - Exports the notes to `backup.json`, then deletes every note without the `line_speed` attribute.
* [ ] `preview.py` - Allows you to preview and edit notes.
  * **DO NOT RUN THIS FILE UNLESS YOU ARE TRYING TO EDIT NOTES.JSON.**
* [ ] `remove_ids.py` - Removes the `id` attribute from every note.
* [ ] `sort_by_time.py` - Sorts every note by their `time` attribute (ascending order).
